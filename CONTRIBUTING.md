# Contributing

Thanks for contributing to mocy.

## Setup

```bash
npm install
npm run typecheck
npm test
```

## Pull requests

- Keep changes focused and well-tested.
- Add or update tests for behavioral changes.
- Run `npm run lint && npm run typecheck && npm test` before opening a PR.

## Commit style

Use clear, imperative commit messages (for example: `Add SQLite pagination index`).
