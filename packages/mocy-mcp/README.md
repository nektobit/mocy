# @mocy/mcp

Official MCP adapter for `mocy`.

## Usage

```bash
npx @mocy/mcp db.json
```

Or via npm script in this workspace:

```bash
npm --workspace packages/mocy-mcp run dev -- db.json
```

## Tools

- `mocy_list_resources`
- `mocy_get_dataset_meta`
- `mocy_get_resource_item`
- `mocy_get_singular_resource`
- `mocy_query_collection`
